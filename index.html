Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo LerAgora coleta, usa e protege as informações dos usuários.

1. Informações Coletadas

  • Informações de login: Nome de usuário e senha (armazenados localmente, sem envio para servidores).
  • Preferências do usuário: Tema claro/escuro, voz preferida, velocidade da leitura, volume.
  • Livros carregados: O aplicativo acessa arquivos PDF locais, mas não envia esses arquivos a servidores.
  • Uso de anúncios: Usamos o Google AdMob para exibir anúncios. O AdMob pode coletar dados conforme sua própria política.

2. Como usamos essas informações

  • Salvar suas preferências de leitura.
  • Exibir corretamente os livros abertos por você.
  • Oferecer anúncios relevantes com AdMob.

3. Compartilhamento de dados

Não compartilhamos nenhuma informação pessoal com terceiros, exceto pelos dados anonimizados coletados via AdMob, conforme suas políticas.

4. Permissões do aplicativo

O LerAgora solicita acesso ao armazenamento apenas para que você possa escolher e abrir arquivos PDF do seu dispositivo.

5. Segurança

Seus dados são armazenados localmente no dispositivo e não são transmitidos para a nuvem ou terceiros.
